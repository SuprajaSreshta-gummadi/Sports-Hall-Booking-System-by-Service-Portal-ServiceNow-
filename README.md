# Sports-Hall-Booking-System-by-Service-Portal-ServiceNow-
Digitize the sports hall booking process using ServiceNow Service Portal to enable fast, transparent, and automated facility reservations.

## Project Objective
In this phase, business requirements for the Sports Hall Booking System were gathered and analyzed. Key functionalities, user roles, and approval workflows were identified. A clear implementation plan and timeline were defined to guide development.


# Enhance User Convenience
*Provide a simple and intuitive booking experience via ServiceNow Service Portal
*Allow users to view available sports halls and time slots in real time
*Reduce manual booking requests and physical coordination
*Enable quick confirmation through automated notifications

# Widgets and Pages Configuration
*A ServiceNow Widgets and Pages Configuration  with the following details:
*Sports Hall / Facility Name
*Sport Type (Badminton, Basketball, Table Tennis,Cricket etc.)
*Booking Date
*Time Slot (Start Time – End Time)
*User Contact Details

# Key Milestones
Requirement Gathering & Analysis
Service Portal UI Design
Availability & Booking Logic Implementation
Approval Workflow Setup
Notification & Email Template Design
User Acceptance Testing (UAT)
Production Deployment
